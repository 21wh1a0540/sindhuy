Face Mask Detector Web Application

Developed a web-based Face Mask Detection system using Flask, which integrates a trained deep learning model to detect the presence of face masks in uploaded images.

Development Environment: Implemented using Python, Flask for the web framework, and Visual Studio Code for development.

Features Implemented:

Deep Learning Model Integration: Integrated a pre-trained Convolutional Neural Network (CNN) model for detecting face masks, which was developed using TensorFlow/Keras.

Image Upload and Prediction: Users can upload images through the web interface, and the application processes the image to predict if the person is wearing a mask or not.

Real-Time Feedback: The application returns real-time feedback on the image classification, indicating whether the person is wearing a mask.

Template Rendering: Utilized Flask’s Jinja templating engine to create dynamic web pages that display results and insights to the user after image processing.

REST API: Implemented a REST API to enable external applications to interact with the face mask detection model and retrieve results via HTTP requests.

Technologies Used: Python, Flask, TensorFlow/Keras, OpenCV, HTML/CSS, Visual Studio Code.
